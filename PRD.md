## 1. 제품 개요

### 한 줄 가치 제안
> **"냉장고 사진 한 장으로, 오늘 도시락 메뉴부터 장보기까지 자동 완성"**

### 제품 설명
**오늘 머먹찌?** 는 냉장고 자동인식 AI 기반 도시락 메뉴 추천 & 식재료 관리 앱입니다.
사진 1장으로 냉장고 재고를 파악하고, 맞춤 메뉴를 추천받고, 구독 배송까지 연결되는 End-to-End 식생활 자동화 서비스입니다.

| 항목 | 내용 |
|------|------|
| 앱 이름 | 오늘 머먹찌? |
| 플랫폼 | iOS / Android |
| 문서 버전 | v2.0 |
| 상태 | Draft |

### 핵심 문제
- 매일 반복되는 메뉴 선정 번거로움 → 단순 배 채우기식 도시락
- 냉장고 재료 파악 어려움 → 중복구매·유통기한 초과로 식비 낭비
- 단순 기능만으론 지속 사용 동기 부족

### 경쟁사 대비 포지셔닝

| 서비스 | 메뉴추천 | 냉장고관리 | 구독배송 |
|--------|----------|------------|----------|
| 만개의레시피 | O | X | X |
| 마켓컬리 | X | X | O |
| 삼성 패밀리허브 | X | O | X |
| **오늘 머먹찌?** | **O** | **O** | **O** |

## 3. 유저 플로우 전체 맵

```
앱 진입
│
├── 홈 탭
│   ├── [오늘의 도시락] 추천 카드 슬라이더
│   │   └── 카드 탭 → 레시피 상세
│   │       ├── 영양 점수 확인
│   │       ├── 식재료 한눈에 보기 (모달)
│   │       ├── 단계별 레시피
│   │       └── "오늘 만들었어요!" CTA
│   └── [나의 도시락 기록] 캘린더
│       ├── 이번 달 절감액 카드
│       └── 기록된 날짜 확인
│
├── 냉장고 탭
│   ├── [빈 상태] → FAB(+) → 식재료 추가
│   └── [재료 있음]
│       ├── 유통기한 경고 배너 (아코디언)
│       ├── 카테고리 필터 탭
│       ├── 식재료 그리드 (5열)
│       └── FAB(+) → 식재료 추가
│           ├── [사진촬영] → 카메라 → 영수증 스캔
│           │   ├── 스캔 중 → 인식 완료
│           │   └── 바텀시트: 발견된 식재료 + 수량 조절
│           ├── [직접입력]
│           │   ├── 검색바 + 카테고리 탭
│           │   ├── 아이콘 그리드 (선택/해제)
│           │   ├── 직접입력 아코디언
│           │   │   ├── 카테고리 바텀시트
│           │   │   └── 아이콘 바텀시트
│           │   └── 선택된 재료 목록 (수량 / 즐겨찾기)
│           └── 담은 재료 확인 → 마이냉장고에 추가하기
│
└── 마이페이지 탭
    ├── 프로필 헤더 (포인트 / 구독현황)
    ├── 도시락 & 기록 / 냉장고 관리 / 구독현황 / 고객지원
    ├── 구독현황
    │   ├── [비구독] → Empty → 구독 시작하기
    │   │   ├── 플랜 선택 (6개)
    │   │   ├── 알레르기 설정 (멀티태그)
    │   │   └── 시작하기
    │   └── [구독 중] → 플랜 변경
    └── 프로필 설정 (설정 아이콘)
        ├── 닉네임 / 이메일 / 알레르기 / 배송정보
        └── 알림설정 토글
```

## 4. 화면별 기능 명세

---

### 4.1 홈 — 오늘의 도시락

**화면 구조**
```
[앱 헤더: 오늘 머먹찌? / 장바구니(뱃지3)]
[탭: 오늘의 도시락 | 나의 도시락 기록]
[인사 섹션]
  - "가나다님," + "오늘도 든든히!" Bold 22px
  - 우측: 햄스터 마스코트 + 문 일러스트
[오늘의 도시락 섹션]
  - 헤더: 날짜(2026.04.16) + "1/3"
  - 가로 스크롤 카드 슬라이더 (2열 프리뷰)
[내가 직접 만들래요! >]
[하단 탭 바: 홈 / 냉장고 / 마이페이지]
```

**도시락 카드 스펙**

| 속성 | 내용 |
|------|------|
| 카드 수 | 최대 3장 |
| 카드 크기 | 약 160px × 200px (2열 프리뷰) |
| 표시 정보 | 도시락명 Bold White, 설명 2줄, "AI 추천메뉴" 태그 |
| 이미지 | 실제 음식 사진 + 하단 그라디언트 오버레이 |
| 인디케이터 | "N/3" 텍스트 |

**기능 목록**
- 사용자 이름 기반 개인화 인사
- 도시락 카드 탭 → 레시피 상세 이동
- 탭 전환: 오늘의 도시락 ↔ 나의 도시락 기록

---

### 4.2 홈 — 나의 도시락 기록 (캘린더)

**기능 목록**
- 이번 달 절감액 = 구독 횟수 × 1회 절감액 (예: 20회 × 7,100원 = 142,000원)
- 이전/다음 월 이동 (< >)
- 기록된 날짜: Primary 색상 점(•) 표시
- 이번달 기록 횟수 (Primary 색상)
- 오늘 날짜: 원형 Primary 배경
- 일요일: Red `#F87167`, 토요일: Blue `#3E97E6`
- 하단: 캘린더 아이콘 + "오늘의 도시락을 기록해주세요!" CTA

---

### 4.3 레시피 상세

**화면 구조**
```
[풀스크린 음식 이미지 헤더 + 좌상단 뒤로가기]
[레시피 타이틀: White Bold 26px, 좌하단]
[도시락 영양 점수 카드]
  - 가로 바: 탄수화물 / 단백질 / 지방 비율
  - 총 kcal + 영양점수 N점 (Primary Bold)
  - "자세히 보기 >" 링크
[햄스터 마스코트 + "같이 만들어봐요" 말풍선]
[이런 식재료가 들어가요! 섹션]
  - "한눈에 보기 >" → 모달
  - 가로 스크롤 식재료 (아이콘 + 이름수량 + X 보유여부)
[레시피 단계: 1~5단계]
[하단 CTA: "오늘 만들었어요!" (활성: Primary / 비활성: Gray)]
``` 

**식재료 한눈에 보기 모달**
- 4열 그리드, 아이콘 + 이름(수량) + × 냉장고 보유 표시
- "(없음)": Gray 텍스트
- `+` 버튼: 없는 재료 추가 가능

### 4.4 냉장고 — 빈 상태 (Empty State)

**화면 구조**
```
[앱 헤더]
[중앙 빈 상태]
  - 생각 말풍선(···) + 햄스터 + 빈 냉장고 일러스트
  - "냉장고가 텅~ 비었어요." 16px Bold
  - "엣, 처음이신가요? 처음 식재료를 입력하면 포인트를 드려요." Caption Gray
[툴팁: "재료를 추가해서 냉장고를 채워봐요" + X 닫기 (Dark Navy BG)]
[FAB: + (Primary 우하단)]
[하단 탭 바: 냉장고 활성]
```

**기능 목록**
- FAB(+) → 식재료 추가 이동
- 툴팁: 최초 진입 1회 표시, X 닫기

### 4.5 냉장고 — 재료 있음

**화면 구조**
```
[앱 헤더]
[유통기한 경고 배너 (아코디언)]
  - 접힘: "▲ 유통기한이 얼마 남지 않은 식재료가 있어요!" + V
  - 펼침: 임박 식재료 아이콘 + 수량뱃지
[햄스터 + 냉장고 + "냉장고 관리는 나한테 맡겨!" 말풍선]
[카테고리 필터 탭 (가로 스크롤)]
  전체 / 아채채소 / 유제품 / 육류 / 수산물 / 과일 / 조미료
[유통기한 색상 안내 텍스트: i 아이콘]
[식재료 그리드 5열]
  - 수량뱃지 좌상단 + 아이콘 + 이름 Caption
  - 마지막: 점선 + 테두리 추가버튼
[FAB: + (Primary 우하단)]
[하단 탭 바]
```

**유통기한 상태 색상**

| 상태 | 뱃지 색상 | 조건 |
|------|-----------|------|
| 넉넉해요 | GrayBlue `#E4E8F4` | 7일 초과 |
| 일주일 남았어요 | Orange `#FFF0E8` | 2~7일 |
| 하루 남았어요 | Red `#FF5555` | 1일 이하 |
| 지났어요 | Black `#000000` | 유통기한 경과 |

### 4.6 식재료 추가 — 진입

**화면 구조**
```
[헤더: 식재료 추가 / < 뒤로 / 홈]
[입력 방식 카드 2개]
  - [사진촬영] AI가 자동인식 해줘요!  (Primary BG 카메라 아이콘)
  - [직접입력] 직접 입력할래요!  (Dark Navy BG 연필 아이콘)
[아코디언: 즐겨찾기 재료]
  - 열림: 재료 행 (아이콘+이름+유통기한+편집) + — + + 검색바
[아코디언: 담은 재료 N]
  - 열림: 5열 아이콘 그리드 + 수량뱃지 + X + 유통기한 색상
  - 최초 열림: 유통기한 색상 안내 툴팁
[하단 CTA: 마이냉장고에 추가하기 (+10P)]
```

**기능 목록**
- 즐겨찾기: swipe-to-delete
- 담은 재료 수 뱃지 표시
- CTA: 담은 재료 0개일 때 비활성

### 4.7 식재료 추가 — 사진촬영 AI 자동인식

**Step 1: 카메라**
```
[검정 카메라 뷰]
[헤더: 자동인식 / < / 플래시]
[스캔 가이드 프레임: Primary 모서리 4개]
["영수증을 촬영해주세요. 식재료를 자동으로 스캔합니다."]
[하단 CTA: "촬영 하기" (White Pill 버튼)]
```

**Step 2: 스캔 중**
```
[영수증 이미지 + 스캔 오버레이]
["식재료를 스캔중입니다."]
["스캔중" 버튼 (비활성)]
```

**Step 3: 인식 완료**
```
[영수증 이미지 + 인식된 항목 Primary Pill 하이라이트]
["N개 식재료를 인식했습니다."]
["스캔완료" 버튼]
```

**Step 4: 발견된 식재료 바텀시트**
```
[바텀시트 핸들]
["발견된 식재료" / "AI가 N개 항목을 인식했습니다."]
[재료 행: 아이콘 + 이름 + 유통기한(편집) + — 수량 +]
[하단 CTA: "+ 재료 담기" (Dark Navy)]
```

**기능 목록**
- 카메라 권한 요청
- 플래시 토글
- 영수증 OCR → 식재료명 자동인식 (AI)
- 유통기한 기본값 제공, 편집 가능
- "+ 재료 담기" → 진입 화면 담은 재료에 추가

---

### 4.8 식재료 추가 — 직접입력

**화면 구조**
```
[헤더: 직접입력 / < / 홈]
[검색바: "찾고싶은 식재료가 있나요?"]
[카테고리 탭 (가로 스크롤, Pill 형태)]
  전체 / 아채채소 / 유제품 / 육류 / 수산물 / 과일 / 조미료
[식재료 아이콘 그리드 6열]
  - 선택: Primary BG (#FFE4D8) 하이라이트
[직접입력 아코디언]
  - [카테고리 V] [아이콘 V] → 바텀시트
  - 이름 / 유통기한 / 수량 입력 필드
  - [입력완료] (Dark Navy)
[선택된 재료 N 아코디언]
  - 아이콘 + 이름 + 유통기한(편집) + — 수량 + / 즐겨찾기별
[하단 CTA: 재료 담기 (Primary)]
```

**카테고리별 식재료 예시**
